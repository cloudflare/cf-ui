# cf-component-page

> Cloudflare Page Component

## Installation

```sh
$ npm install cf-component-page
```

## Usage

```jsx
import React from 'react';
import { Page, PageHeader } from 'cf-component-page';

const PageComponent = () => (
  <Page>
    <PageHeader title="Title" subtitle="Subtitle" />
  </Page>
);

export default PageComponent;
```
